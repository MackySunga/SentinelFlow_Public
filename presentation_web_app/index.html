SentinelFlow

FFT-Enhanced Deep Learning for Network Intrusion Detection

A final project prototype that demonstrates how NetFlow traffic can be cleaned, transformed into time-windowed signals, enhanced using Fast Fourier Transform, and evaluated through binary and multiclass intrusion detection.

NetFlowFFTDeep LearningDetection

Problem Statement

Traditional flow features may miss hidden attack rhythms.

Intrusion Detection Systems often rely on time-domain traffic features such as packet counts, byte counts, flow duration, and throughput. These features are useful, but they may not clearly expose burst patterns or repeated behavior found in Distributed Denial-of-Service, brute-force, botnet, and port-scanning attacks.

Challenge 1Large NetFlow datasets require organized preparation.
Challenge 2Time-domain features may miss repeated traffic patterns.
Challenge 3Models need clear metrics beyond accuracy.

Final Project Summary

SentinelFlow demonstrates the feasibility of combining signal processing and deep learning for intrusion detection.

The prototype does not need to claim production-level deployment. Its purpose is to show a working, explainable pipeline: data preparation, FFT-enhanced traffic representation, model evaluation, confusion matrices, and a dashboard-centered cybersecurity presentation.
