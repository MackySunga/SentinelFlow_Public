FFT-Enhanced Intrusion Detection Prototype

SentinelFlow Web

A Django-based prototype for demonstrating NetFlow upload, traffic profiling, FFT feature extraction, and binary plus multiclass intrusion detection workflows.

Upload Dataset View Dashboard

1. NetFlow Data

Upload CSV, TSV, TXT, or Parquet files containing flow timestamps, packets, bytes, duration, labels, and attack classes.

2. FFT Profiling

Traffic is grouped into time windows and converted into frequency-domain indicators such as spectral energy and burstiness.

3. Detection Dashboard

Use notebook-trained models and generated reports to present binary and multiclass detection results.

Prototype Scope

This app is designed for final project demonstration. Training still happens in Jupyter notebooks. The Django app focuses on upload, validation, profiling, dashboard viewing, and report presentation.
